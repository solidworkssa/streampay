# StreamPay

Continuous payment streaming protocol.

## Features
- Multi-chain support (Base & Stacks)
- Secure Contract Logic
- React/Next.js Frontend

## Contracts
- Base: `contracts/base/src/StreamPay.sol`
- Stacks: `contracts/stacks/contracts/streampay.clar`

## Setup
```bash
pnpm install
pnpm dev
```
